# AdventJS - 2025

AdventJS es un proyecto creado por **Midudev**.

Está inspirado en el Adviento, el tiempo litúrgico cristiano, con retos de programación en **JavaScript**, **TypeScript** y **Python**. Donde cada día se habilita un nuevo desafío, con un total de **24 retos**.

## Requisitos

- Gestor de paquetes: [pnpm](https://pnpm.io), [npm](https://www.npmjs.com) o [yarn](https://yarnpkg.com)

## Instalación

Clona el repositorio y entra en la carpeta:

```bash
git clone https://github.com/Martiks38/adventjs-2025.git
```

Instala las dependencias:

```bash
npm i
```

```bash
pnpm install
```

```bash
yarn install
```
